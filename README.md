# DIP Calculator – Internal POC

## Overview
The DIP Calculator is a small internal web-based tool created as a proof of concept to support operational date calculations related to DIP timelines.

## Purpose
- Reduce manual date calculation errors  
- Speed up routine operational checks  
- Provide a clear visual indication of whether a DIP is required  

## How it works
- The user enters an expiration date
- The calculation can be performed using days, months, or years
- The result clearly indicates:
  - OK
  - CREATE A DIP

## Technical Notes
- Static application (HTML, CSS, JavaScript)
- No backend
- No database
- No authentication
- No data storage
- No API calls
- Runs entirely in the client browser

## Deployment
Currently hosted on GitHub Pages for proof-of-concept purposes.

## Status
Proof of Concept – Internal use only.
